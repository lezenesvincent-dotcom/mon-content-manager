✅ Embedded Shared Database (Méthode manuelle)

Data is stored directly in the HTML file - No external services needed!

🚀 How to share content across machines:
  1. Create/edit content on any machine
  2. Click the "📥 EXPORT DATA" button below
  3. The data will be copied to your clipboard and logged to console
  4. Open the HTML file in a text editor
  5. Find the section: const DEFAULT_CONTENT_STORE = [
  6. Paste your data array between the brackets
  7. Save the file and push to GitHub
  8. 🎉 All machines loading this page will see the data!
💡 How it works:
  • Embedded storage: Data is stored directly in the HTML file
  • Local additions: New content is saved in localStorage
  • Merge on load: Both sources are combined when the page loads
  • 100% offline: No external API calls needed
  • GitHub sync: Push updated file to share with all machines
